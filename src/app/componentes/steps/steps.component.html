Instalar PrimeNg y font-awesome:

npm install primeng font-awesome --save Agregar los estilos en el archivo .angular-cli.json:

styles: [
"../node_modules/font-awesome/css/font-awesome.min.css",
"../node_modules/primeng/resources/themes/omega/theme.css",
"../node_modules/primeng/resources/primeng.min.css"
] Importar las animaciones(en app.modules.ts) que son necesarias para ciertos componentes primeng:

import {{ BrowserAnimationsModule }} from @angular/platform-browser/animations'; Agregar el modulo al array imports:

imports: [
BrowserModule,
BrowserAnimationsModule
]
